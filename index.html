About this project

This (side-) project curates resources for web-animation, used for interaction or webdesign. I cuurently dont include mobile animation related resources.

I made this Side-project list after finding lots of resources on the web and finding that i can pull content from a google sheet.

you can comment or suggest in the public google sheet here. Tools Ive used to make it : isotope, images-loaded, cloudinary's url2png, this pen for getting data from google sheet, pace.js, Feather icons
